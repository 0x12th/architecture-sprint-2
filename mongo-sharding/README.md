# Шардирование

### Запуск проекта

```shell
docker compose up -d
```

### Подключение к серверу конфигурации и инициализация

```shell
./scripts/config-server-init.sh
```


### Инициализация шардов

```shell
./scripts/shards-init.sh
```


### Инициализация роутера и наполнение его тестовыми данными

```shell
./scripts/router-init.sh
```
